Jammit - Location-Based Social App for Musicians
Overview

Jammit is a social Android application designed to connect musicians seeking jam partners, band members, or a local music community.

This project showcases full-stack mobile development expertise, focusing on architectural design and the implementation of complex, real-time features.

Project Status: Portfolio Piece (Alpha/Beta stage).



Key Features:
- Real-Time Live Chat: Implemented a robust, low-latency messaging service using Firebase Realtime Database/Firestore to enable instant communication between matched users.
- 
- Geo-Location Search: Utilizes GeoLocation and Geohash algorithms for efficient and accurate filtering of users based on geographical proximity.
- 
- Detailed User Profiles: Allows users to create and manage rich profiles, including instruments, preferred genres, and skill levels.

- Modern Navigation: Employs the AndroidX Navigation Component for intuitive and maintainable screen flow management.

- User Authentication: Secure user sign-up and sign-in implemented using Firebase Authentication.
